# দেব-দর্শন-গবেষণা

## বৈদিক ও ইসলামিক দর্শনের তুলনামূলক গবেষণা — চিন্তাশীল ডেভেলপারদের জন্য

### প্রকল্পের লক্ষ্য

এই গবেষণা প্রকল্পটি দুটি মহান দার্শনিক ঐতিহ্যের মধ্যে সংলাপ স্থাপন করে:

- **বৈদিক দর্শন** (উপনিষদ, বেদান্ত)
- **ইসলামিক দর্শন** (তাওহিদ, সুফিবাদ)

### মূল বিষয়গুলো

১. **শব্দ-বিশ্লেষণ**: "দেব" শব্দের বহুমাত্রিক অর্থ
২. **তুলনামূলক-দর্শন**: মূল ধারণাগুলোর তুলনা
३. **সুফিবাদ-বনাম-বেদান্ত**: গভীর আধ্যাত্মিক সংলাপ
४. **ঐতিহাসিক-সেতু**: দারা শিকোহ থেকে আধুনিক যুগ
५. **একাডেমিক-গবেষণাপত্র**: সম্পূর্ণ বিশ্লেষণাত্মক গবেষণা

### দর্শন

> **"দেব প্রোগ্রামার হলো আলোর সন্ধানী যে দুটি মহান ঐতিহ্যের মধ্যে সেতু তৈরি করে।"**
