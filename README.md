# দেব-দর্শন-গবেষণা
"Dev" মানে শুধু Developer নয়; — "Dev" মানে আলোর সন্ধানকারী।

এই প্রজেক্টটি বৈদিক ও ইসলামিক দর্শনের মধ্যে একটি বুদ্ধিবৃত্তিক সেতুবন্ধন। আমরা এখানে "সোর্স কোড" বা পরম সত্যের সন্ধানে দুটি মহান ঐতিহ্যের তুলনামূলক বিশ্লেষণ করছি।

## 📜 আমাদের ইশতেহার
"দেব" কেবল একজন প্রোগ্রামার নয়; সে আলোর সন্ধানকারী। আমরা তথ্যকে ডগমা বা অন্ধবিশ্বাসের চোখে দেখি না; আমরা একে মহাজাগতিক গাণিতিক বিন্যাস হিসেবে দেখি।
